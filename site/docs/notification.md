## Notification
### Basic
Define notification menu.

:::
```js
    render() {
        return (
            <Notification reverse>
                <ListItem>
                   Message 1
                </ListItem>
                <ListItem>
                   Message 2
                </ListItem>
                <ListItem>
                   Message 3
                </ListItem>
            </Notification>
        );
    }

```
:::

#### API

| Props | Description | Default Value |
| ------------- | ----------- | ----------- |
| className | Component className. | null |
| style | Component style. | null |
